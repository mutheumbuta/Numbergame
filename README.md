# Numbergame 

The Number Game is a simple Python terminal game where the computer randomly generates a number between 1 and 100, and the player tries to guess it within a limited number of attempts.

The game provides hints such as "Too High" or "Too Low" to help the player guess the correct number.

## Features

* Easy – 10 attempts

* Medium – 7 attempts

* Hard – 5 attempts

* Feedback after each guess

* Displays remaining attempts

* Ends when the player guesses correctly or runs out of attempts

## Technologies Used

1. Python 3

2. Python random module

## Getting started

### Prerequisites

1. Make sure you have Python 3 installed on your computer.

2. Check installation with:
python --version

### Installation

Clone the repository:
git clone https://github.com/mutheumbuta/Numbergame.git

Navigate to the project folder:
cd Numbergame

Run the game:
guessgame.py


## How the game works
 
* The program generates a random number between 1 and 100.

* The user selects a difficulty level.

* The player guesses the number.

* The program gives hints:

* Too High

* Too Low

* The game continues until:

* The correct number is guessed, or

* The player runs out of attempts.
